Analysis Endpoint

POST /analyze

Upload a soil image to receive a full chemical and physical analysis report.

Request Headers

Content-Type: multipart/form-data

Request Body

Param Type Description
image File The soil sample image (JPG/PNG).

  "status": "success",
  "soil_type": "Black Soil",
  "confidence": "94.2%",
  "ph_min": 6.5,
  "nitrogen": "52 mg/kg",
  "recommended_crops": ["Cotton", "Wheat"],
  "health_score": "88/100"
}
